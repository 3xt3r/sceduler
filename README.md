В офисе установлена IP-камера видеонаблюдения. У нее есть веб-интерфейс управления, но он доступен только из локальной сети. В камере есть уязвимость: человек, который подключен к той же сети Wi-Fi, может отправить специальный запрос и просматривать весь видеопоток без авторизации. 
Выберите верный вектор CVSS для данной уязвимости.
Расчет CVSS выполняется на сайте: https://bdu.fstec.ru/calc3.
