Верный ответ: B
CVSS:3.1/AV:A/AC:L/PR:N/UI:N/S:U/C:H/I:N/A:N
CVSS Score: примерно 6.5 Medium

Пояснение к верному вектору
AV:A — Adjacent / Смежная сеть
Атакующий должен находиться в той же локальной сети или Wi-Fi сети, что и IP-камера. Уязвимость нельзя эксплуатировать напрямую из интернета, поэтому это не AV:N.

AC:L — Low / Низкая сложность
Для эксплуатации достаточно отправить специальный запрос. Не нужны редкие условия, сложная последовательность действий или нестабильный сценарий.

PR:N — None / Привилегии не требуются
Атакующему не нужна учетная запись в веб-интерфейсе камеры. Просмотр видеопотока возможен без авторизации.

UI:N — None / Взаимодействие пользователя не требуется
Эксплуатация не требует действий со стороны другого пользователя. Атакующий сам отправляет запрос к камере.

S:U — Unchanged / Область воздействия не изменяется
Уязвимость находится в IP-камере и влияет на данные этой же камеры — видеопоток. Перехода на другой компонент безопасности нет.

C:H — High / Высокое влияние на конфиденциальность
Атакующий получает доступ ко всему видеопотоку камеры. Это серьезное раскрытие конфиденциальной информации, особенно для офисной камеры видеонаблюдения.

I:N — None / Влияния на целостность нет
По условию атакующий только просматривает видеопоток. Он не изменяет настройки камеры, не подменяет видео и не управляет устройством.

A:N — None / Влияния на доступность нет
Эксплуатация не отключает камеру, не ломает сервис и не мешает другим пользователям получать доступ к видеопотоку.

Почему остальные варианты неверные
A неверный, потому что AV:N означает эксплуатацию через сеть, например через интернет. В условии сказано, что доступ возможен только из локальной сети.
C неверный, потому что AV:L означает локальную эксплуатацию на самом устройстве или системе. Здесь атакующий действует по сети, находясь рядом в Wi-Fi.
D неверный, потому что AC:H завышает сложность. В условии нет специальных сложных условий, нужен только специальный запрос.
E неверный, потому что PR:L означает, что атакующему нужна учетная запись с низкими правами. В условии сказано, что авторизация не требуется.
F неверный, потому что UI:R означает необходимость взаимодействия пользователя. В данном случае атакующий сам отправляет запрос, без участия жертвы.
